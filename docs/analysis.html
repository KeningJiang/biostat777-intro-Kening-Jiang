Research question

To describe and compare the Measles, Mumps, and Rubella (MMR) vaccine rates and overall vaccine rates among public and private schools in California and Ohio

Why vaccinating children is important

Intended audience

Public health practitioners, schools, and parents in California and Ohio

Original data

Immunization rate data for schools in the US, as compiled by The Wall Street Journal. Modified dataset was included in TidyTuesday.

Data dictionary

https://github.com/rfordatascience/tidytuesday/blob/master/data/2020/2020-02-25/readme.md#data-dictionary

Data loading

Show the code 
# Load the packages
library("tidyverse")
library("tidytuesdayR")
library("here")

# tests if a directory named "data" exists locally
if (!dir.exists(here("data"))) {
    dir.create(here("data"))
}

if (!file.exists(here("data", "tuesdata_measles.RDS"))) {
    tuesdata <- tidytuesdayR::tt_load("2020-02-25")

    # save the files to RDS objects
    saveRDS(tuesdata$measles, file = here("data", "tuesdata_measles.RDS"))
}

# Load the datasets
measles <- readRDS(here("data", "tuesdata_measles.RDS"))

Data wrangling

Show the code 
measles_clean<- 
  measles %>%
  mutate(type = replace(type, which(is.na(type)), "Other")) %>%
  filter(mmr>=0) %>%
  filter(overall>=0) %>%
  select(-c('xrel','xmed', 'xper','lat','lng')) %>%
  filter(state=="California" | state=="Ohio")
    
measles_summary <- 
  measles_clean %>%
  group_by(state, type)  %>%
  summarize (
             n=n(),
             mean_mmr = mean(mmr),
             min_mmr = min(mmr),
             max_mmr = max(mmr),
             mean_overall = mean(overall),
             min_overall = min(overall),
             max_overall = max(overall)
             )
Pay Attention

Substantial missingmess of variables like school type. Only California and Ohio were analyzed due to relatively complete data.

# Define a custom theme to start with
custom_theme <- theme(
  panel.background = element_blank(),
  axis.ticks = element_blank(),
  axis.title = element_text(size=14, face="bold"),
  axis.line = element_line(size = 0.5),
  plot.title = element_text(color="black", size=18, face="bold", hjust=0.5),

figure1 <- ggplot(data=measles_summary, aes(x=state, y=n, fill=type)) +
           geom_bar(stat="identity") +
           scale_fill_manual(name = "School type",
